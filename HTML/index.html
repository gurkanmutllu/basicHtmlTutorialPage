Links

The HTML <a> tag defines a hyperlink.
It has the following syntax:
<a>href="url">link text</a>
If we want to open it in another tab;
<a href="url" target="_blank">link text</a>

Image Size - Width and Height

We can use the style attribute to specify the width and height of an image.

Lists

There are three ways to making a list:
  1. Ordered List
    HTML Tags Outputs
    <ol type="1">Drinks
    <li>Coffee</li>
    <li>Tea</li>
    <li>Water</li>
    </ol>
      Drinks
    1. Coffee
    2. Tea
    3. Water

    The variable type determines the type of the ordered list.
    type: {1, a, A, I, i}

  2. Unordered List
    HTML Tags Outputs
    <ul type="bullet">Foods
    <li>Egg</li>
    <li>Soup</li>
    <li>Steak</li>
    </ul>
      Foods
    • Egg
    • Soup
    • Steak

  3. Definition List
    HTML Tags Outputs
    <dl>
    <dt>Dog</dt>
    <dd>-A dog is a cute pet.</dd>
    <dt>Lion</dt>
    <dd>-A lion is a dangerous wild animal.</dd>
    </dl>
    Dog
    -A dog is a cute pet.
    Lion
    -A lion is a dangerous wild animal.
